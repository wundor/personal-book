# personal-book
Book of your financial jorney

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ docker-compose up -d # This will launch the database
$ npm run start # After that you can hit the app on port 4000

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
